# Architecture & Components

KoamiStarterKit is built on a modern, robust stack designed for developer experience and performance.

## UI Components (Shadcn Vue)

The starter kit includes a comprehensive suite of UI components from **Shadcn Vue** to jumpstart your development. These components are accessible, customizable, and drop-in ready.

### Form Elements
- `Input`, `Select`, `Checkbox`, `Radio Group`
- `Switch`, `Slider`, `Textarea`
- `Form`, `Combobox`

### Feedback
- `Alert`, `Badge`, `Progress`
- `Skeleton`, `Spinner`
- `Sonner` (Toast notifications)
- `Tooltip`

### Overlay
- `Dialog` (Modals)
- `Drawer`, `Sheet`
- `Popover`, `Hover Card`
- `Context Menu`, `Dropdown Menu`

### Layout
- `Card`, `Aspect Ratio`
- `Resizable`, `Scroll Area`
- `Separator`

### Navigation
- `Sidebar`, `Navigation Menu`
- `Breadcrumb`, `Tabs`
- `Menubar`, `Pagination`, `Stepper`

### Data Display
- `Table`, `Avatar`
- `Accordion`, `Collapsible`
- `Carousel`, `Calendar`

## Project Structure

The project follows standard Laravel 12 conventions but includes specific directories for the frontend stack:

- `resources/js/`: Contains the Vue 3 application and Inertia pages.
- `resources/js/Components/ui/`: Where your Shadcn components live.
- `app/Filament/`: Contains the Admin Panel resources and configuration.
