# Installation

Get KoamiStarterKit running locally in under 5 minutes.

## Prerequisites

- **PHP 8.2+** with extensions: `ctype`, `curl`, `dom`, `fileinfo`, `filter`, `hash`, `mbstring`, `openssl`, `pcre`, `pdo`, `session`, `tokenizer`, `xml`
- **Composer** - PHP dependency manager
- **Node.js LTS** - JavaScript runtime
- **Git** - Version control

## Method 1: Laravel Installer (Recommended)

```bash
# Install Laravel CLI if you haven't
composer global require laravel/installer

# Create project
laravel new my-app --using=koamishin/koamistarterkit
cd my-app

# Run setup wizard
php artisan setup:starter-kit
```

## Method 2: Composer

```bash
composer create-project koamishin/koamistarterkit my-app
cd my-app
php artisan setup:starter-kit
```

## Method 3: Git Clone (Contributors)

```bash
git clone https://github.com/koamishin/KoamiStarterKit.git my-app
cd my-app
composer install
cp .env.example .env
php artisan key:generate
touch database/database.sqlite
php artisan migrate
npm install
php artisan setup:starter-kit
```

## What the Setup Wizard Does

The `setup:starter-kit` command personalizes your project:

- **Composer.json** - Updates author name, email, package name
- **GitHub Actions** - Configures CI/CD workflows for your repo
- **Docker** - Sets up container registry (Docker Hub or GHCR)

> **Note:** The wizard is interactive. Just answer the prompts and it handles the configuration files automatically.

## Verify Installation

```bash
# Start all services (Laravel, Vite, Queue, Logs)
composer run dev

# In another terminal, run tests
composer test
```

Visit `http://localhost:8000` to see your app.

## Default Login

After installation, seed the database to create a default admin:

```bash
php artisan db:seed
```

Then login with:
- **Email**: `admin@example.com`
- **Password**: `password`

> **Warning:** Change the default password immediately after first login.
