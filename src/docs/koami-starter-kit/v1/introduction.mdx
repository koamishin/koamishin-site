# Introduction

KoamiStarterKit is a production-ready Laravel 12 starter kit that gets you from zero to deployed in minutes, not days.

## What You Get

- **Complete Auth System** - Login, registration, 2FA, email verification, password reset
- **Admin Dashboard** - Filament-powered admin panel with user management  
- **Role-Based Access** - Pre-configured Spatie permissions (Admins vs Users)
- **Modern Frontend** - Vue 3 + Inertia.js + Tailwind CSS 4 + TypeScript
- **Type-Safe Routing** - Wayfinder generates TypeScript definitions from Laravel routes
- **Production Pipeline** - Docker, GitHub Actions CI/CD, automated releases
- **Code Quality** - Pre-configured Pint, PHPStan, Rector, and Pest tests

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Laravel 12, Octane, Fortify |
| Frontend | Vue 3, Inertia.js v2, Tailwind CSS 4 |
| Admin | FilamentPHP |
| Database | SQLite (dev) / PostgreSQL or MySQL (prod) |
| Testing | Pest PHP |

## Quick Start

```bash
# Create new project
composer create-project koamishin/koamistarterkit my-app
cd my-app

# Run setup wizard
php artisan setup:starter-kit

# Start development
composer run dev
```

Visit `http://localhost:8000` - you're ready to build.
