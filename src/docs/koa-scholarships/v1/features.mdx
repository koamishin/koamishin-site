# Features

KoaScholarships offers a rich set of features tailored for both students and administrators.

## For Students 🎓

### Smart Matching
The system automatically filters scholarships based on your profile (School Type, GPA, etc.), so you only see opportunities you're eligible for.

### Streamlined Workflow
- **Step-by-step Guidance**: Clear instructions from draft to submission.
- **Real-time Status**: Track your application status (Pending, Approved, Rejected).
- **Document Management**: Easily upload requirements and handle resubmissions with feedback.

### Profile & Service
- **Profile Management**: Keep your academic and personal details up to date.
- **Community Service**: Log service hours and submit reports directly through the portal.

## For Administrators 🛡️

### Program Management
- **Dynamic Slots**: Automatic calculation of available slots based on `Total Budget / Per Student Budget`.
- **Custom Requirements**: Define specific criteria for each scholarship program.

### Comprehensive Workflow
1. **Review**: Verify uploaded documents.
2. **Eligibility**: Auto-check GPA and unit requirements.
3. **Enrollment**: Manage accepted students.
4. **Service Monitoring**: Track required community service days.
5. **Disbursement**: Process financial aid batches.

### Financial Tracking
Monitor disbursements and view remaining budget per program in real-time.
